1. Descarregar i convertir dades hor√†ries de contaminaci√≥ atmosf√®rica

Les dades autom√†tiques hor√†ries de contaminaci√≥ atmosf√®rica de diferents poblacions es descarreguen del servidor XVPCA del Departament de Medi Ambient i Sostenibilitat de la Generalitat de Catalunya i les converteixes en format compatible amb openair amb el codi Python seg√ºent dissenyat per fer-ho servir al teu Google colab.

Primer far√†s un an√†lisi anual de Martorell entre 1991 i 2024, posteriorment realitzar√†s un altre estudi plurianual d'una localitat catalana.

    
    
import pandas as pd
from google.colab import files

# Pujar el fitxer CSV
print("Carrega el teu fitxer CSV:")
uploaded = files.upload()

# Carregar el fitxer CSV
file_name = list(uploaded.keys())[0]
df = pd.read_csv(file_name)

# Comprovar les columnes necess√†ries
required_columns = ['data', 'contaminant', 'h01', 'h02', 'h03', 'h04', 'h05', 'h06', 'h07', 'h08',
                    'h09', 'h10', 'h11', 'h12', 'h13', 'h14', 'h15', 'h16', 'h17', 'h18', 'h19',
                    'h20', 'h21', 'h22', 'h23', 'h24']
if not all(col in df.columns for col in required_columns):
    raise ValueError("El fitxer no cont√© les columnes requerides.")

# Expandir les hores en files amb un format de temps
df = df.melt(id_vars=['data', 'contaminant'],
             value_vars=[f"h{i:02}" for i in range(1, 25)],
             var_name='hora',
             value_name='valor')

# Convertir la columna 'hora' en un format d'hora real
df['hora'] = df['hora'].str.extract('(\d+)').astype(int) - 1  # Convertir 'h01', 'h02', ... a 0-23
df['data'] = pd.to_datetime(df['data'], errors='coerce') + pd.to_timedelta(df['hora'], unit='h')

# Eliminar la columna 'hora' i reestructurar les dades
df = df.drop(columns=['hora'])

# Seleccionar nom√©s les columnes necess√†ries (data, contaminant i les hores expandides)
df = df[['data', 'contaminant', 'valor']]

# Pivotar el DataFrame per tenir una columna per cada contaminant
pivot_df = df.pivot(index='data', columns='contaminant', values='valor').reset_index()

# Ordenar per data
pivot_df = pivot_df.sort_values(by='data')

# Convertir les dates a format amb espai (en lloc de T) per ISO abans de guardar el CSV
pivot_df['data'] = pivot_df['data'].dt.strftime('%Y-%m-%d %H:%M:%S')

# Renombrar la columna 'data' com 'date'
pivot_df = pivot_df.rename(columns={'data': 'date'})

# Convertir els noms dels contaminants a min√∫scules
pivot_df.columns = [col.lower() if isinstance(col, str) and col != 'date' else col for col in pivot_df.columns]

# Substituir els valors buits (NaN) per 'NA'
pivot_df = pivot_df.fillna('NA')

# Guardar el fitxer CSV processat
output_file = 'processed_data.csv'
pivot_df.to_csv(output_file, index=False)

# Descarregar el fitxer processat
files.download(output_file)
print(f"El fitxer processat s'ha desat com: {output_file}")

2. Descarregar dades meteorol√≤giques semihor√†ries a format compatible amb openair

Hem de descarregar dades meteorol√≤giques del Meteocat, concretament els parametres 30 i 31, que corresponen a velocitat de vent (ws) i direcci√≥ del vent (wd) del servidor XEMA amb dades semihor√†ries del Meteocat triant el codi de l'estaci√≥ meteorol√≤gica m√©s propera a la nostra estaci√≥ mesuradora de contaminaci√≥ per poder combinar dades de contaminaci√≥ i metereol√≤giques segons codi compartit amb Github del professor.


   
import pandas as pd  # Importem la llibreria pandas per treballar amb dades tabulars
import numpy as np  # Importem numpy per gestionar valors num√®rics i operacions matem√†tiques
from google.colab import files  # Importem el m√≤dul files per pujar i descarregar fitxers a Google Colab

# Pujar un fitxer CSV des de l'ordinador
uploaded = files.upload()  # Mostra una finestra per seleccionar i pujar un fitxer

# Obtenir el nom del fitxer que s'ha pujat
# uploaded √©s un diccionari on les claus s√≥n els noms dels fitxers pujats
filename = list(uploaded.keys())[0]  # Agafem el primer (i √∫nic) fitxer pujat

# Llegir el fitxer CSV i carregar-lo en un DataFrame de pandas
# encoding='latin1' s'utilitza per evitar problemes amb car√†cters especials
# df ser√† la nostra taula de dades


df = pd.read_csv(filename, encoding='latin1')

# Convertim la columna 'DATA_LECTURA' a un format de data i hora
# format='%d/%m/%Y %I:%M:%S %p' indica com est√† estructurada la data al fitxer original
# errors='coerce' fa que els valors incorrectes es converteixin en NaT (valors buits de tipus data)
df['DATA_LECTURA'] = pd.to_datetime(df['DATA_LECTURA'], format='%d/%m/%Y %I:%M:%S %p', errors='coerce')

# Filtrar nom√©s les files on els minuts de la columna DATA_LECTURA siguin 0
# Aix√≤ vol dir que ens quedem nom√©s amb les dades corresponents a hores senceres (per exemple, 10:00, 11:00, etc.)
df = df[df['DATA_LECTURA'].dt.minute == 0]

# Ordenar les dades per data (no es fa expl√≠citament, per√≤ es pressuposa que els passos seg√ºents ho gestionen)

# Obtenim el nom de la antepen√∫ltima columna (la tercera des del final)
antepenultima_columna = df.columns[-3]  # Exemple: podria ser 'VALOR'

# Obtenim el nom de la tercera columna des del final
tercera_columna = df.columns[-6]  # Exemple: podria ser 'PAR√ÄMETRE MESURAT'

# Creem una nova columna 'ws'
# Posem el valor de la antepen√∫ltima columna nom√©s si la tercera columna val 30
# np.where(condici√≥, valor_si_cert, valor_si_fals) permet aplicar una condici√≥ fila per fila
df['ws'] = np.where(df[tercera_columna] == 30, df[antepenultima_columna], np.nan)

# Creem una altra nova columna 'wd'
# Posem el valor de la antepen√∫ltima columna nom√©s si la tercera columna val 31
df['wd'] = np.where(df[tercera_columna] == 31, df[antepenultima_columna], np.nan)

# Ens quedem nom√©s amb tres columnes: DATA_LECTURA, ws i wd
df = df[['DATA_LECTURA', 'ws', 'wd']]

# Canviem el nom de la columna 'DATA_LECTURA' a 'date' per facilitar la comprensi√≥
df.columns = ['date', 'ws', 'wd']

# Agrupem les dades per data i agafem el primer valor de cada grup
# Aix√≤ assegura que nom√©s hi hagi una fila per cada data √∫nica
df = df.groupby('date', as_index=False).first()

# Definim el nom del fitxer de sortida
output_filename = "filtered_data.csv"

# Guardem les dades processades en un nou fitxer CSV sense la columna d'√≠ndex
df.to_csv(output_filename, index=False)

# Descarreguem el fitxer CSV generat
files.download(output_filename)

# Mostrem un missatge indicant que el proc√©s ha finalitzat
print("Proc√©s completat! S'ha descarregat el fitxer filtered_data.csv")
