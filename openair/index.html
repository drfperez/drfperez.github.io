1. Descarregar i convertir dades horÃ ries de contaminaciÃ³ atmosfÃ¨rica

Les dades automÃ tiques horÃ ries de contaminaciÃ³ atmosfÃ¨rica de diferents poblacions es descarreguen del servidor XVPCA del Departament de Medi Ambient i Sostenibilitat de la Generalitat de Catalunya i les converteixes en format compatible amb openair amb el codi Python segÃ¼ent dissenyat per fer-ho servir al teu Google colab.

import pandas as pd
from google.colab import files

# Pujar el fitxer CSV
print("Carrega el teu fitxer CSV:")

# Carregar el fitxer CSV

# Comprovar les columnes necessÃ ries
required_columns = ['data', 'contaminant', 'h01', 'h02', 'h03', 'h04', 'h05', 'h06', 'h07', 'h08',
                    'h09', 'h10', 'h11', 'h12', 'h13', 'h14', 'h15', 'h16', 'h17', 'h18', 'h19',

             value_vars=[f"h{i:02}" for i in range(1, 25)],
             var_name='hora',
             value_name='valor')

pivot_df = pivot_df.sort_values(by='data')

# Convertir les dates a format amb espai (en lloc de T) per ISO abans de guardar el CSV

# Renombrar la columna 'data' com 'date'
pivot_df = pivot_df.rename(columns={'data': 'date'})

# Convertir els noms dels contaminants a minÃºscules

# Substituir els valors buits (NaN) per 'NA'
pivot_df = pivot_df.fillna('NA')

# Guardar el fitxer CSV processat
output_file = 'processed_data.csv'
pivot_df.to_csv(output_file, index=False)

# Descarregar el fitxer processat
files.download(output_file)

2. Descarregar dades meteorolÃ²giques semihorÃ ries a format compatible amb openair

Hem de descarregar dades meteorolÃ²giques del Meteocat, concretament els parametres 30 i 31, que corresponen a velocitat de vent (ws) i direcciÃ³ del vent (wd) del servidor XEMA amb dades semihorÃ ries del Meteocat triant el codi de l'estaciÃ³ meteorolÃ²gica mÃ©s propera a la nostra estaciÃ³ mesuradora de contaminaciÃ³ per poder combinar dades de contaminaciÃ³ i metereolÃ²giques segons codi compartit amb Github del professor.


   
import pandas as pd  # Importem la llibreria pandas per treballar amb dades tabulars
import numpy as np  # Importem numpy per gestionar valors numÃ¨rics i operacions matemÃ tiques
from google.colab import files  # Importem el mÃ²dul files per pujar i descarregar fitxers a Google Colab

# Pujar un fitxer CSV des de l'ordinador
uploaded = files.upload()  # Mostra una finestra per seleccionar i pujar un fitxer

# Obtenir el nom del fitxer que s'ha pujat
# uploaded Ã©s un diccionari on les claus sÃ³n els noms dels fitxers pujats
filename = list(uploaded.keys())[0]  # Agafem el primer (i Ãºnic) fitxer pujat

# Llegir el fitxer CSV i carregar-lo en un DataFrame de pandas
# encoding='latin1' s'utilitza per evitar problemes amb carÃ cters especials
# df serÃ  la nostra taula de dades


df = pd.read_csv(filename, encoding='latin1')

# Convertim la columna 'DATA_LECTURA' a un format de data i hora
# format='%d/%m/%Y %I:%M:%S %p' indica com estÃ  estructurada la data al fitxer original
# errors='coerce' fa que els valors incorrectes es converteixin en NaT (valors buits de tipus data)
df['DATA_LECTURA'] = pd.to_datetime(df['DATA_LECTURA'], format='%d/%m/%Y %I:%M:%S %p', errors='coerce')

# Filtrar nomÃ©s les files on els minuts de la columna DATA_LECTURA siguin 0
# AixÃ² vol dir que ens quedem nomÃ©s amb les dades corresponents a hores senceres (per exemple, 10:00, 11:00, etc.)
df = df[df['DATA_LECTURA'].dt.minute == 0]

# Ordenar les dades per data (no es fa explÃ­citament, perÃ² es pressuposa que els passos segÃ¼ents ho gestionen)

# Obtenim el nom de la antepenÃºltima columna (la tercera des del final)
antepenultima_columna = df.columns[-3]  # Exemple: podria ser 'VALOR'

# Obtenim el nom de la tercera columna des del final
tercera_columna = df.columns[-6]  # Exemple: podria ser 'PARÃ€METRE MESURAT'

# Creem una nova columna 'ws'
# Posem el valor de la antepenÃºltima columna nomÃ©s si la tercera columna val 30
# np.where(condiciÃ³, valor_si_cert, valor_si_fals) permet aplicar una condiciÃ³ fila per fila
df['ws'] = np.where(df[tercera_columna] == 30, df[antepenultima_columna], np.nan)

# Creem una altra nova columna 'wd'
# Posem el valor de la antepenÃºltima columna nomÃ©s si la tercera columna val 31
df['wd'] = np.where(df[tercera_columna] == 31, df[antepenultima_columna], np.nan)

# Ens quedem nomÃ©s amb tres columnes: DATA_LECTURA, ws i wd
df = df[['DATA_LECTURA', 'ws', 'wd']]

# Canviem el nom de la columna 'DATA_LECTURA' a 'date' per facilitar la comprensiÃ³
df.columns = ['date', 'ws', 'wd']

# Agrupem les dades per data i agafem el primer valor de cada grup
# AixÃ² assegura que nomÃ©s hi hagi una fila per cada data Ãºnica
df = df.groupby('date', as_index=False).first()

# Definim el nom del fitxer de sortida
output_filename = "filtered_data.csv"

# Guardem les dades processades en un nou fitxer CSV sense la columna d'Ã­ndex
df.to_csv(output_filename, index=False)

# Descarreguem el fitxer CSV generat
files.download(output_filename)

# Mostrem un missatge indicant que el procÃ©s ha finalitzat
print("ProcÃ©s completat! S'ha descarregat el fitxer filtered_data.csv")
