# Play Chess
♔ ♚

An interactive web application for playing chess against an advanced AI using Stockfish, featuring a modern and responsive interface. Built with HTML, CSS, and JavaScript, this app allows users to play games, load/save PGN files, navigate moves, and more.

## Description
Play Chess is an online chess game where you can play against a powerful AI based on Stockfish (expert level) or a simpler AI for lower difficulties. It includes features like board flipping, undoing moves, loading historical games (such as the Immortal Game with comments), and handling pawn promotions via a modal dialog.

The application is optimized for both mobile and desktop devices, with a dark-themed, elegant design.

## Features
- **Play Against AI:** Difficulty levels from 1 to 6 (1-5: Simple minimax AI, 6: Stockfish engine).
- **Intuitive Interface:** Buttons for flipping the board, undoing moves, starting new games, saving/loading PGN files, and navigating through move history.
- **Promotion Handling:** Modal for selecting promotion pieces (Queen, Rook, Bishop, Knight).
- **Historical Games:** Load and view commented games like the Immortal Game.
- **Responsive Design:** Works seamlessly on mobile and desktop with touch support.
- **Engine Status:** Displays the active engine (Stockfish or local).
- **Navigation Controls:** Go to start, previous, next, or end of the game.
