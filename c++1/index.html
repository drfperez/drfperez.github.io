C++ amb jocs

Aquest curs inclourà la creació de quatre jocs diferents que ajuden a aprendre els conceptes bàsics de programació en C++. Aquests jocs són:

  • Timberman: Un joc de plataformes que inclou elements com enemics, armes ràpides, pickups i un personatge que es pot nivellar.
  • Pong: Un joc clàssic de tennis electrònic que inclou col·lisió, sons i condicions de finalització.
  • Zombie Shooter: Un joc de tir en primera persona que inclou col·lisió, sons espacialitzats i condicions de finalització.
  • Run: Un joc de plataformes que inclou elements com un fons de shader realista, parallax scrolling, sons espacialitzats i un personatge animat.

Elements de codi en C++

Escriure el codi per aquests jocs ajudarà a aprendre els següents elements de C++:

  • Variables, operadors i decisions: Comprendre com C++ emmagatzema i manipula dades.
  • Cicles, matrius i funcions: Comprendre com utilitzar cicles, matrius i funcions per implementar mecàniques de joc.
  • Col·lisió, sons i condicions de finalització: Comprendre com implementar col·lisió, sons i condicions de finalització en els jocs.
  • Objecte-Orientat i SFML: Comprendre com utilitzar objecte-Orientat i SFML per crear jocs interactius.

Avantatges de C++ sobre Python o C#

C++ té alguns avantatges sobre Python o C# per crear jocs:

  • Ràpida execució: C++ és conegut per la seva velocitat i eficiència, el que és important per a jocs que requereixen una alta velocitat de processament.
  • Control baix: C++ ofereix un control baix sobre els recursos de hardware, el que és útil per a jocs que requereixen una gran quantitat de recursos.
  • Compatibilitat amb múltiples plataformes: C++ es pot compilar i executar en múltiples plataforms, incloent Windows, macOS i Linux.
  • Comunitat activa: C++ té una comunitat activa i rica en recursos, el que facilita la resolució de problemes i l'aprenentatge.
  • Indústria estàndard: C++ és un llenguatge estàndard de l'indústria dels jocs, el que facilita la col·laboració amb altres desenvolupadors i l'enteniment de codi existent.

En resum, C++ és un llenguatge molt útil per a la creació de jocs, especialment per a aquells que busquen una alta velocitat de processament i un control baix sobre els recursos de hardware.
